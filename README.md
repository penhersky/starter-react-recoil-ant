# React + TypeScript + Vite + Recoil

### Quick start

Step 1

```
pnpm install
```

Step 2
Add .env from .env.example

Step 3

```
pnpm dev:api
```

### Run only Web app

```
pnpm dev
```

### Run Mock Sever

```
yarn mock-server
```

### Build project

```
yarn build
```

### Build project

```
yarn build
```
